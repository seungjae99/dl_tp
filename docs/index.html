Car Classification with CNN

A lightweight convolutional neural network trained from scratch — no transfer learning — to classify four Korean sedan and SUV models from images, achieving 97% validation accuracy on approximately 800 training images.

Images were resized with aspect-ratio-preserving scaling and zero-padded to a fixed 300x200 canvas. A 15% validation split was carved from training data at runtime. Train/test split uses a fixed seed (42) for reproducibility.

Model Architecture

  • INPUT200 x 300 x 3 RGB, rescaled to [0, 1]
  • BLOCK 1Conv2D (32 filters, 3x3, ReLU) — BatchNormalization — MaxPooling2D — L2
  • BLOCK 2Conv2D (64 filters, 3x3, ReLU) — BatchNormalization — MaxPooling2D — L2
  • HEADGlobalAveragePooling2D — Dense (128, ReLU, L2) — Dropout (0.6) — Dense (4, Softmax)

GlobalAveragePooling replaces a flatten layer, reducing parameters and improving spatial invariance. The two-block design keeps the model compact while capturing sufficient texture and shape features to distinguish visually similar vehicles.

Key Findings

  • 01A custom two-block CNN without transfer learning achieves 97% validation accuracy on a small (~800 image) 4-class dataset, demonstrating that a well-regularized shallow architecture can handle the task without pretrained weights.
  • 02GlobalAveragePooling combined with Dropout (0.6) and L2 regularization on all trainable layers effectively suppresses overfitting: train/val accuracy converge to within 0.5 percentage points.
  • 03K5 precision (0.71 on test) is the dominant failure mode. The car shares body proportions with Santa Fe and Sorento — classes that blur visually at this dataset scale. Expanding dataset diversity or adding viewpoint augmentation is the highest-leverage next step.
